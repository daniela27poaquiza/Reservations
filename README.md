# Reservations
This bot makes reservations for game spots at anytime you will like too. 
We expect our bot to identify the amount of tables available and game avaliable. 
Our bot takes notice of the time for organizational purposes.
To use this bot the individual has to ask the bot 'what game and what table is avaliable' and then the information will appear.
After the person has decided what they want the just have to type in the game and table they want.
